# Femtography Plotter

Mesh search and plotting code for UVA-CNF group

## Instructions
- Clone this repo
- Create `venv` virtual environment
    ```
    virtualenv venv 
    ```
- Run
    ```
    source venv/bin/activate
    pip install -r requirements.txt
    python app.py
    ```
